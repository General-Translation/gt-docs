## Overview

This guide is a step by step tutorial on how to translate strings in your Next.js app
using General Translation using [translation dictionaries](/docs/next/reference/dictionaries/dictionaries).

We will also discuss how to add variables to your strings.

## Setup

### Prerequisites

We assume that you already have installed `gt-next` in your project and have followed the [Quick Start Guide](/docs/next/tutorials/quickstart).

### Steps

<Steps>
  <Step>
    **Add the `initGT()` plugin**

    In your `next.config.mjs` file, add the [`initGT()` plugin](/docs/next/api/config/initGT) to your Next.js app.
    This allows `gt-next` to access your translation dictionary.

    ```js title="next.config.mjs" copy
    import { initGT } from 'gt-next/config';

    const withGT = initGT();

    export default withGT({});

    ```
  </Step>
  <Step>
  **Create a dictionary file**

 This file will store all of the string content that you want to translate.

  ```jsx title="src/dictionary.jsx" copy
    const dictionary = {
      id: 'here is a string' // [!code highlight]
    }
    export default dictionary;
  ```
  </Step>
  <Step>
  **Reference the string in your component**

  Now that you have centralized all of the content you want to translate,
  you can reference the string in your component.

  <Tabs items={["Client Components", "Server Components"]}>
    <Tab value="Client Components">

      Client components use the [`useGT()`](/docs/next/api/dictionary/useGT) hook to access the translation function `t()`.

      ```jsx title="src/components/MyComponent.jsx" copy
      import { useGT } from 'gt-next/client';
      export default function MyComponent() {
        const { t } = useGT();
        return (
          <div>
            <h1>{t('id')}</h1> // [!code highlight]
          </div>
        );
      }
      ```

    </Tab>
    <Tab value="Server Components">

      Server components use the [`getGT()`](/docs/next/api/dictionary/getGT) function to access the translation function `t()`.

      ```jsx title="src/pages/index.jsx" copy
      import { getGT } from 'gt-next/server';
      export default function Home() {
        const t = await getGT();
        return (
          <div>
            <h1>{t('id')}</h1> // [!code highlight]
          </div>
        );
      }
      ```
    </Tab>
  
  </Tabs>
  </Step>
</Steps>

You have now internationalized your strings!
Try [loading your app in a different language](/docs/next/tutorials/quickstart#see-your-app-in-a-different-language) and watch the magic happen.

## Adding Variables

You can add variables to your strings through a pattern very similar to string formatting.

Inside of your dictionary file, add an identifier where you want to insert the variable.

```jsx title="src/dictionary.jsx" copy
// [!code word:{identifier}]
const dictionary = {
  id: ' here is a string with a variable: {identifier} '
}
export default dictionary;
```

Then, in your component, pass the variable as a second argument to the `t()` function.

```jsx title="src/components/MyComponent.jsx" copy
// [!code word:identifier]
import { useGT } from 'gt-next/client';
export default function MyComponent() {
  const { t } = useGT();
  return (
    <div>
      <h1>{t('id', { identifier : 'variable content' })}</h1> // [!code highlight]
    </div>
  );
}
```

## Notes
 * In order to use the dictionary pattern, you must add the [`initGT()` plugin](/docs/next/api/config/initGT).
 * Reference the string content using [`useGT()`](/docs/next/api/dictionary/useGT) for client components and [`getGT()`](/docs/next/api/dictionary/getGT) for server components.
