## Übersicht

Die `<LocaleSelector>` Komponente wird verwendet, um die Spracheinstellung des Benutzers auszuwählen.
Es ist eine clientseitige Komponente, die ein Dropdown-Menü zur Auswahl der Spracheinstellung bereitstellt.

## Referenz

### Rückgaben

Eine Komponente, die es dem Benutzer ermöglicht, seine Sprache auszuwählen.

---

## Beispiele

### Grundlegende Verwendung

```jsx
"use client";
import { LocaleSelector } from 'gt-next/client';

export default function MyComponent() {
    return (
        <LocaleSelector />
    );
}
```

---

## Notizen
* Die `<LocaleSelector>` Komponente ermöglicht es Ihnen, eine andere Sprache für Ihre App auszuwählen.
* Die `<LocaleSelector>` Komponente ist in der Serverkomponente nicht verfügbar.
