## Übersicht

Der `useSetLocale` Hook wird verwendet, um die Spracheinstellung des Benutzers festzulegen.
Dies ist nur clientseitig und muss in eine [`<GTProvider>`](/docs/next/api/components/gtprovider) Komponente eingebettet werden.


## Referenz

### Rückgaben

Eine Funktion, die die Spracheinstellung des Benutzers festlegt.

---

## Beispiele

### Grundlegende Verwendung

```jsx
import { useSetLocale } from 'gt-next/client';

export default function MyComponent() {
    const setLocale = useSetLocale();

    return (
        <button onClick={() => setLocale('en')}>Sprache festlegen</button>
    );
}
```
