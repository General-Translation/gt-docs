## Überblick

Die `<T>`-Komponente ist die primäre Methode zur Übersetzung von Inhalten in General Translation.
Sie kann zusammen mit Wörterbüchern oder eigenständig verwendet werden.

Die `<T>`-Komponente kann zur Übersetzung von Text, HTML und JSX verwendet werden.

Alles, was Sie tun müssen, ist Ihren Text in die `<T>`-Komponente einzuwickeln, und er wird übersetzt.

Keine zusätzliche Konfiguration erforderlich!

```jsx
import { T } from 'gt-react'; // [!code highlight]

export default function MyComponent() {
  return (
    // [!code highlight]
    <T>
      <h1>List of Shakespeare Plays: </h1>
      <ul>
        <li>Hamlet</li>
        <li>A Midsummer Night's Dream</li>
        <li>Macbeth</li>
      </ul>
    </T> // [!code highlight]
  );
}
```


### Unser Rat

Wir empfehlen die Verwendung der `<T>`-Komponente aufgrund ihrer Einfachheit, besonders wenn Sie neu in der Internationalisierung (i18n) sind.
Wir bieten [Wörterbücher](/docs/next/reference/dictionary-reference) für diejenigen an, die dieses Designmuster aus früheren Erfahrungen bevorzugen oder die Integration in bestehende Codebasen erleichtern möchten.
