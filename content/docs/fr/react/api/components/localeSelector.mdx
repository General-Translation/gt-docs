## Vue d'ensemble

Le composant `<LocaleSelector>` est utilisé pour sélectionner la langue de l'utilisateur.
C'est un composant côté client qui fournit un menu déroulant pour sélectionner la langue.

## Référence

### Retours

Un composant qui permet à l'utilisateur de sélectionner sa langue.

---

## Exemples

### Utilisation de base

```jsx
import { LocaleSelector } from 'gt-react';

export default function MyComponent() {
    return (
        <LocaleSelector />
    );
}
```

---

## Notes
* Le composant `<LocaleSelector>` vous permet de sélectionner une langue différente pour votre application.
* Le composant `<LocaleSelector>` n'est pas disponible dans le composant serveur.
