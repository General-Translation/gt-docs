## Aperçu

Toutes nos bibliothèques sont open source et gratuites. Elles fonctionnent de manière totalement autonome, prêtes à l'emploi.

Vous pouvez les utiliser dans n'importe quel projet, sans jamais avoir besoin de créer un compte ou de vous inscrire aux services de General Translation.

De manière autonome, nos bibliothèques prennent en charge les fonctionnalités suivantes :

- Encapsulation de composants React entiers pour la traduction
- Dictionnaires
- Configuration facile du projet
- Middleware

L'utilisation de General Translation comme fournisseur de traduction débloquera les fonctionnalités suivantes :

- Traduction à la demande
- Traductions automatiques par IA
- Un CDN de traduction sans tracas

## Gestion des traductions

Pour générer les fichiers de traduction pour votre projet, exécutez l'outil CLI avec la commande suivante :

<Tabs items={['Next.js', 'React']}>
  <Tab value="Next.js">
    ```bash
    npx gtx-cli init
    npx gtx-cli generate
    ```
  </Tab>
  <Tab value="React">
    ```bash
    npx gtx-cli init
    npx gtx-cli generate
    ```
  </Tab>
</Tabs>

Cette commande générera les fichiers de données JSON pour le code source de votre projet. 
Vous pouvez ensuite utiliser ces fichiers avec votre propre fournisseur de traduction.
Pour charger les traductions dans `gt-next` ou `gt-react`, utilisez la fonction `loadTranslations`, comme spécifié dans
le guide des traductions locales.

Consultez la [référence de l'API CLI](/docs/cli/generate) pour plus d'informations.
