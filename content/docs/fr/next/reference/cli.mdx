## Aperçu

Ceci est un document de référence sur l'outil CLI de Traduction Générale.

L'outil CLI est utilisé pour gérer les traductions de votre projet.
Il vous permet de [générer des traductions](/docs/cli/translate), [générer des modèles de traduction](/docs/cli/generate), et [configurer votre projet pour la traduction](/docs/cli/setup).
En savoir plus sur `gtx-cli` [ici](/docs/cli).

## Que fait l'outil CLI ?

Si vous utilisez `gt-next`, vous serez principalement concerné par les commandes [`translate`](/docs/cli/translate) et [`setup`](/docs/cli/setup).
L'outil CLI est utilisé pour gérer votre projet et vos traductions.

 * [La commande `translate`](/docs/cli/translate) est destinée à être ajoutée au pipeline CD.
   Elle est utilisée pour analyser votre projet à la recherche de contenu à traduire.
   Le contenu est ensuite téléchargé sur un CDN ou ajouté au bundle de votre application pour être utilisé lors de l'exécution.
 * [La commande `setup`](/docs/cli/setup) est utilisée pour intégrer automatiquement la bibliothèque General Translation dans votre projet.
   Elle analysera votre projet et enveloppera tout contenu JSX avec un composant `<T>`.
