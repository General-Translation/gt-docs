## Aperçu

Le hook `useSetLocale` est utilisé pour définir la locale de l'utilisateur.
Ceci est uniquement côté client, et il doit être enveloppé dans un composant [`<GTProvider>`](/docs/next/api/components/gtprovider).


## Référence

### Retours

Une fonction qui définit la langue de l'utilisateur.

---

## Exemples

### Utilisation de base

```jsx
import { useSetLocale } from 'gt-next/client';

export default function MyComponent() {
    const setLocale = useSetLocale();

    return (
        <button onClick={() => setLocale('en')}>Définir la langue</button>
    );
}
```
