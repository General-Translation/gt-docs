## Introduction

Félicitations pour la configuration de votre projet NextJS ! Maintenant, préparons-le pour la traduction en utilisant `gt-next`.

Actuellement, votre projet est un projet normal, non internationalisé, qui ne peut être lu que dans une seule langue.
Cependant, avant de commencer la traduction, nous devons ajouter la bibliothèque à votre projet et configurer votre environnement.

### Installer `gt-next`
Ensuite, ajoutons la bibliothèque `gt-next` à votre projet. Vous pouvez le faire en exécutant la commande suivante :

```bash copy
npm i gt-next
```

### Configurer votre environnement
Créez un fichier `.env` à la racine de votre projet et ajoutez votre clé API et l'identifiant de votre projet.

```env copy
GT_API_KEY="YOUR_GT_API_KEY"
GT_PROJECT_ID="YOUR_GT_PROJECT_ID"
```

Si vous n'avez pas encore configuré votre projet, connectez-vous simplement à votre [compte General Translation](https://wwww.generaltranslation.com/signin).
Ouvrez la page des clés API *DEV* et créez une nouvelle clé API *DEV*.
Enfin, copiez la clé API *DEV* et l'identifiant du projet et collez-les dans votre fichier `.env`.

## Conclusion
À présent, tout devrait être prêt et configuré pour commencer la traduction.
Tout ce que nous avons à faire maintenant est d'ajouter le composant `<T>` à notre projet et de commencer à traduire !
