## Aperçu

L'outil CLI de Traduction Générale (`gtx-cli`) vous permet de traduire automatiquement votre projet dans n'importe quelle [langue prise en charge](/docs/reference/supported-locales).

Il offre une prise en charge de première classe pour [`gt-next`](/docs/react) et [`gt-react`](/docs/react) avec votre projet, vous permettant de scanner votre projet pour le contenu traduisible, traduire votre projet et générer des fichiers JSON de données de traduction.

L'outil CLI peut également être utilisé pour générer des traductions pour des bibliothèques i18n tierces comme [next-intl](https://next-intl.dev/) ou [i18next](https://react.i18next.com/).

---

Selon la bibliothèque i18n utilisée par votre projet, veuillez suivre l'un des tutoriels suivants :

* [gt-next](/docs/cli/tutorials/gt)
* [gt-react](/docs/cli/tutorials/gt)
* [next-intl](/docs/cli/tutorials/next-intl-quickstart)
* [i18next](/docs/cli/tutorials/i18next-quickstart)
* [Autre](/docs/cli/tutorials/json)




## Utilisation

| Commande | Description |
| --- | --- |
| [`npx gtx-cli init`](/docs/cli/init) | (Optionnel) Configure les paramètres GT de votre projet. |
| [`npx gtx-cli setup`](/docs/cli/setup) | Guide d'installation étape par étape qui prépare votre projet pour la traduction. |
| [`npx gtx-cli scan`](/docs/cli/scan) | Analyse votre projet pour le contenu traduisible. Peut être utilisé à plusieurs reprises pour mettre à jour votre projet. |
| [`npx gtx-cli translate`](/docs/cli/translate) | Traduit votre projet via l'API GT. |
| [`npx gtx-cli generate`](/docs/cli/generate) | Génère un nouveau fichier JSON de données de traduction. Doit être utilisé lors de l'utilisation autonome de la bibliothèque. |
