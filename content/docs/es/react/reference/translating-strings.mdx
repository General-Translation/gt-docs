## Visión general

Esta guía es un tutorial paso a paso sobre cómo traducir cadenas en tu aplicación React
con [`useGT()`](/docs/react/api/strings/useGT).


## Configuración

### Requisitos previos

Asumimos que ya has instalado `gt-react` en tu proyecto y has seguido o estás siguiendo la [Guía de Inicio Rápido](/docs/react/tutorials/quickstart).

### Traducción de Cadenas
Para cualquier cadena, utiliza [`useGT()`](/docs/react/api/strings/useGT).
Recuerda que `useGT()` debe ser llamado dentro de un componente hijo de [`<GTProvider>`](/docs/react/api/components/gtprovider).

```jsx title="src/components/MyComponent.jsx" copy
import { useGT } from 'gt-react';

export default function MyComponent() {
  const t = useGT(); // [!code highlight]
  return (
    <div>
      <h1>{t('This is a string that gets translated')}</h1> // [!code highlight]
    </div>
  );
}
```

### Añadiendo Variables
Las variables son valores que pueden cambiar, pero no se traducen.
Para añadir variables a tus cadenas, utiliza el siguiente patrón:

```jsx title="MyComponent.jsx" copy
import { useGT } from 'gt-react';

export default function MyComponent() {
  const t = useGT(); // [!code highlight]
  return (
    <div>
      <h1>{t('Hello there, {username}', { variables: { username: 'Brian123' }})}</h1> // [!code highlight]
    </div>
  );
}
```

---

## Notas
 * Para la traducción de cadenas, use [`useGT()`](/docs/react/api/strings/useGT).
 * Las variables se pueden agregar a las cadenas usando el patrón `{ variables: { key: value } }`.
