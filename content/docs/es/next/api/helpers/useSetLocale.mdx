## Resumen

El hook `useSetLocale` se utiliza para establecer la configuración regional del usuario.
Esto es solo del lado del cliente, y debe estar envuelto en un componente [`<GTProvider>`](/docs/next/api/components/gtprovider).


## Referencia

### Retornos

Una función que establece la configuración regional del usuario.

---

## Ejemplos

### Uso Básico

```jsx
import { useSetLocale } from 'gt-next/client';

export default function MyComponent() {
    const setLocale = useSetLocale();

    return (
        <button onClick={() => setLocale('en')}>Establecer Localización</button>
    );
}
```
