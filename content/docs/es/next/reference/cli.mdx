## Descripción general

Este es un documento de referencia sobre la herramienta CLI de Traducción General.

La herramienta CLI se utiliza para gestionar las traducciones de tu proyecto.
Te permite [generar traducciones](/docs/cli/translate), [generar plantillas de traducción](/docs/cli/generate), y [configurar tu proyecto para traducción](/docs/cli/setup).
Lee más sobre `gtx-cli` [aquí](/docs/cli).

## ¿Qué hace la herramienta CLI?

Si estás utilizando `gt-next`, principalmente te interesarán los comandos [`translate`](/docs/cli/translate) y [`setup`](/docs/cli/setup).
La herramienta CLI se utiliza para gestionar tu proyecto y tus traducciones.

 * [El comando `translate`](/docs/cli/translate) está diseñado para ser añadido al pipeline de CD.
   Se utiliza para escanear tu proyecto en busca de contenido que necesite ser traducido.
   El contenido se sube luego a una CDN o se añade al paquete de tu aplicación para ser utilizado en tiempo de ejecución.
 * [El comando `setup`](/docs/cli/setup) se utiliza para integrar automáticamente la biblioteca General Translation en tu proyecto.
   Escaneará tu proyecto y envolverá cualquier contenido JSX con un componente `<T>`.
