## Descripción general

Todas nuestras bibliotecas son de código abierto y de uso gratuito. Funcionan completamente independientes, listas para usar.

Puedes utilizarlas en cualquier proyecto, sin necesidad de crear una cuenta o registrarte en los servicios de General Translation.

De forma independiente, nuestras bibliotecas admiten las siguientes características:

- Envolver componentes completos de React para traducción
- Diccionarios
- Configuración sencilla de proyectos
- Middleware

Usar General Translation como proveedor de traducción desbloqueará las siguientes características:

- Traducción bajo demanda
- Traducciones automáticas con IA
- Una CDN de traducción sin complicaciones

## Gestión de traducciones

Para generar los archivos de traducción para tu proyecto, ejecuta la herramienta CLI con el siguiente comando:

<Tabs items={['Next.js', 'React']}>
  <Tab value="Next.js">
    ```bash
    npx gtx-cli init
    npx gtx-cli generate
    ```
  </Tab>
  <Tab value="React">
    ```bash
    npx gtx-cli init
    npx gtx-cli generate
    ```
  </Tab>
</Tabs>

Este comando generará los archivos de datos JSON para el código fuente de tu proyecto. 
Luego puedes usar estos archivos con tu propio proveedor de traducción.
Para cargar las traducciones en `gt-next` o `gt-react`, utiliza la función `loadTranslations`, como se especifica en
la guía de traducciones locales.

Consulta la [referencia de la API CLI](/docs/cli/generate) para obtener más información.
