## Visión general

Este tutorial te mostrará cómo gestionar automáticamente los archivos de traducción de tu proyecto en [`next-intl`](https://next-intl.dev/).

Seguiremos estos 4 pasos:
<Steps>
  <Step>
    Añade tus variables de entorno
  </Step>
  <Step>
    Instala [`gtx-cli`](/docs/cli)
  </Step>
  <Step>
    Inicializa tu proyecto con el comando [`gtx-cli init`](/docs/cli/init)
  </Step>
  <Step>
    Ejecuta [`gtx-cli translate`](/docs/cli/translate#translate-json)
  </Step>
</Steps>

<Callout>
  **Consejo:**
  Evita las molestias de usar archivos de traducción con el [componente `<T>`](/docs/usage/t).
</Callout>

---

## Paso 1: Añade tus variables de entorno

Añade tu clave API de producción y el ID del proyecto a tus variables de entorno.
Esto es necesario para usar la herramienta `gtx-cli`.
Puedes obtener estos datos desde el [panel de control de General Translation](https://generaltranslation.com/dashboard).

```bash title=".env"
GT_API_KEY=<your-api-key>
GT_PROJECT_ID=<your-project-id>
```

## Paso 2: Instala `gtx-cli`

Instala la herramienta `gtx-cli` en tu proyecto.

<Tabs items={['npm', 'yarn', 'bun', 'pnpm']}>
  <Tab value="npm">
  ```bash 
  npm i --save-dev gtx-cli
  ``` 
  </Tab>
  <Tab value="yarn">
  ```bash 
  yarn add --dev gtx-cli
  ```
  </Tab>

  <Tab value="bun">
  ```bash 
  bun add --dev gtx-cli
  ```
  </Tab>
  <Tab value="pnpm">
  ```bash 
  pnpm add --save-dev gtx-cli
  ```
  </Tab>
</Tabs>

## Paso 3: Inicializa tu proyecto con el comando `gtx-cli init`

Ejecuta el comando `gtx-cli init` para configurar tu proyecto.

```bash
npx gtx-cli init
```

## Paso 4: Añade el comando `gtx-cli translate` a tu proceso de compilación

Añade el comando `gtx-cli translate` a tu proceso de compilación o CI antes del comando de compilación para agregar automáticamente traducciones a tu proyecto.

```json title="package.json"
{
  "scripts": {
    "translate": "npx gtx-cli translate",
    "build": "npm run translate && <your build command>"
  }
}
```
Esto generará traducciones para todos tus idiomas y las guardará en tu proyecto.
Si quieres confirmar estos archivos en tu repositorio, puedes ejecutar este comando antes de hacer commit.


¡Has terminado! Ahora tu proyecto actualizará automáticamente todos tus archivos JSON de traducción cada vez que tu proyecto cambie.

## Notas
 * Puedes añadir traducciones automáticamente a tu proyecto con el comando [`gtx-cli translate`](/docs/cli/translate#translate-json).
 * Si quieres confirmar tus archivos de traducción, puedes ejecutar el comando `gtx-cli translate` antes de hacer commit.
