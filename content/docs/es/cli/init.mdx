## Uso

Utiliza este comando para inicializar los ajustes de GT de tu proyecto. Este comando es opcional y solo sirve para crear un archivo `gt.config.json` en la raíz de tu proyecto.

```bash
npx gtx-cli init
```

## Descripción general

El comando `init` te ayuda a configurar los ajustes de GT para tu proyecto.

Creará un archivo `gt.config.json` en la raíz de tu proyecto.

El archivo contendrá los siguientes ajustes:

- `defaultLocale`: El idioma predeterminado para tu proyecto.
- `locales`: Un array de idiomas compatibles para tu proyecto.
- `files`: Este es un objeto que contiene información sobre el contenido que deseas traducir.

Para obtener información más específica sobre el archivo `gt.config.json`, consulta la [documentación de configuración](/docs/cli/reference/config).
