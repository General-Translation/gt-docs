## Overview

The `<Num>` component renders a formatted number string, providing options for customization such as a name, default value, and formatting options.
The number is formatted based on the current locale and any optional formatting options passed.

This is useful for rendering numbers with proper localization, such as currency, percentages, or custom formats.
Additionally, the contents of the `<Num>` component will remain private.

### Returns

`JSX.Element` containing the formatted number as a string.

## Example

The `<Num>` component can be used to format numbers with localization and optional customization.
It must be used inside of a `<T>` component or in a dictionary entry.

```jsx title="QuantityDisplay.jsx" copy
import { T, Num } from 'gt-next'; // [!code highlight]

/**
 * Renders the quantity of an item in the inventory.
 */
export default function Inventory(item) {
    return (
        <T>
            There are currently <Num> {item.quantity} </Num> units in stock.  // [!code highlight]
        </T>
    );
}
```

For more details and usage examples of the `<Num>` component and other variable components like `<Currency>`, `<DateTime>`, and `<Var>`,
see the [Using Variable Components](/docs/next2/reference/using-variable-components) documentation.
