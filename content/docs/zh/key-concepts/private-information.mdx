## 概述
私人信息通常是指任何应该保持私密的信息。
例如，用户的社会安全号码（SSN）、税务识别号（TIN）或电子邮件地址。

为了在翻译过程中保持这些信息的私密性，我们使用`<Var>`组件。
`<Var>`组件用于包装任何不应该被共享的内容，
而`<Var>`组件内的内容将始终保持不变。
这对于姓名、电子邮件、地址等非常有用。

有时，我们确实想要显示私人信息，但仍需要根据用户的首选语言、地区法规等重新格式化它。
例如，我们可能想要以用户的本地货币显示其银行账户余额。
在这些情况下，我们改用`<Currency>`、`<DateTime>`或`<Num>`组件。


### 示例
`<Var>`组件对于显示私人信息也很有用。

```jsx title="PrivateInfo.jsx" copy
import { T, Var } from 'gt-next'

export default function PrivateInfo(email) {
    return (
        <T id='private-info'>
            Your email address is <Var>{email}</Var>.
        </T>
    );
}
```
`<Var>`组件的子元素不会被翻译。
其内容永远不会传递给通用翻译API。
其内容不会分发到CDN。
