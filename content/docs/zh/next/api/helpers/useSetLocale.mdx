## 概述

`useSetLocale` 钩子用于设置用户的语言环境。
这仅限于客户端，并且必须包装在 [`<GTProvider>`](/docs/next/api/components/gtprovider) 组件中。


## 参考

### 返回

一个设置用户语言环境的函数。

---

## 示例

### 基本用法

```jsx
import { useSetLocale } from 'gt-next/client';

export default function MyComponent() {
    const setLocale = useSetLocale();

    return (
        <button onClick={() => setLocale('en')}>设置语言环境</button>
    );
}
```
