## 介绍

恭喜你成功设置了你的 NextJS 项目！现在，让我们使用 `gt-next` 准备好进行翻译。

目前你的项目是一个普通的、非国际化的项目，只能用一种语言阅读。
然而，在我们开始翻译之前，我们需要将库添加到你的项目中并设置你的环境。

### 安装 `gt-next`
接下来，让我们将 `gt-next` 库添加到你的项目中。你可以通过运行以下命令来完成此操作：

```bash copy
npm i gt-next
```

### 设置你的环境
在你的项目根目录下创建一个 `.env` 文件，并添加你的 api key 和 project id。

```env copy
GT_API_KEY="YOUR_GT_API_KEY"
GT_PROJECT_ID="YOUR_GT_PROJECT_ID"
```

如果你还没有设置你的项目，只需登录到你的 [General Translation 账户](https://wwww.generaltranslation.com/signin)。
打开 *DEV* API Keys 页面并创建一个新的 *DEV* API key。
最后，复制 *DEV* API key 和 project ID 并将它们粘贴到你的 `.env` 文件中。

## 结论
到现在为止，一切都应该准备就绪，可以开始翻译了。
我们现在要做的就是将 `<T>` 组件添加到我们的项目中并开始翻译！
