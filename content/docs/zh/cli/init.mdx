## 用法

使用此命令初始化您项目的GT设置。此命令是可选的，仅用于在项目根目录创建`gt.config.json`文件。

```bash
npx gtx-cli init
```

## 概述

`init`命令帮助您配置项目的GT设置。

它将在您项目的根目录创建一个`gt.config.json`文件。

该文件将包含以下设置：

- `defaultLocale`：您项目的默认语言环境。
- `locales`：您项目支持的语言环境数组。
- `files`：这是一个包含您想要翻译的内容信息的对象。

有关`gt.config.json`文件的更具体信息，请参阅[配置文档](/docs/cli/reference/config)。
