## 概述

通用翻译 CLI 工具（`gtx-cli`）允许您将项目自动翻译成任何[支持的语言](/docs/reference/supported-locales)。

它为您的项目提供了与 [`gt-next`](/docs/react) 和 [`gt-react`](/docs/react) 的一流支持，使您能够扫描项目中的可翻译内容，翻译您的项目，并生成翻译数据 JSON 文件。

CLI 工具还可用于为第三方 i18n 库生成翻译，如 [next-intl](https://next-intl.dev/) 或 [i18next](https://react.i18next.com/)。

---

## 使用方法

| 命令 | 描述 |
| --- | --- |
| [`npx gtx-cli init`](/docs/cli/init) | （可选）配置您项目的 GT 设置。 |
| [`npx gtx-cli setup`](/docs/cli/setup) | 逐步设置指南，为您的项目准备翻译。 |
| [`npx gtx-cli scan`](/docs/cli/scan) | 扫描您的项目中的可翻译内容。可以重复使用以更新您的项目。 |
| [`npx gtx-cli translate`](/docs/cli/translate) | 通过 GT API 翻译您的项目。 |
| [`npx gtx-cli generate`](/docs/cli/generate) | 生成新的翻译数据 JSON 文件。在单独使用库时应使用。 |
