## Overview

The `gt-next-cli` is a command-line interface (CLI) for managing your Next.js application's translations.


## Installation

To install the `gt-next-cli`, run the following command:

```bash copy
npm i gt-next-cli
```

## Usage

| Command | Description |
| --- | --- |
| [`npx gt-next-cli setup`](/docs/next-cli/setup) | Automatically sets up your project for translation. |
| [`npx gt-next-cli translate`](/docs/next-cli/translate) | Translates your project into the supported locales. |
