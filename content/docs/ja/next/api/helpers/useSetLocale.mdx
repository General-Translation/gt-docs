## 概要

`useSetLocale` フックは、ユーザーのロケールを設定するために使用されます。
これはクライアントサイドのみで、[`<GTProvider>`](/docs/next/api/components/gtprovider) コンポーネントでラップする必要があります。


## リファレンス

### 戻り値

ユーザーのロケールを設定する関数。

---

## 例

### 基本的な使用法

```jsx
import { useSetLocale } from 'gt-next/client';

export default function MyComponent() {
    const setLocale = useSetLocale();

    return (
        <button onClick={() => setLocale('en')}>ロケールを設定</button>
    );
}
```
