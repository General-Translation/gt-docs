## はじめに

NextJSプロジェクトのセットアップおめでとうございます！次に、`gt-next`を使用して翻訳の準備をしましょう。

現在、あなたのプロジェクトは通常の非国際化プロジェクトであり、1つの言語でしか読むことができません。
しかし、翻訳を始める前に、ライブラリをプロジェクトに追加し、環境を設定する必要があります。

### `gt-next`のインストール
次に、`gt-next`ライブラリをプロジェクトに追加しましょう。以下のコマンドを実行することでこれを行うことができます：

```bash copy
npm i gt-next
```

### 環境の設定
プロジェクトのルートに`.env`ファイルを作成し、APIキーとプロジェクトIDを追加します。

```env copy
GT_API_KEY="YOUR_GT_API_KEY"
GT_PROJECT_ID="YOUR_GT_PROJECT_ID"
```

まだプロジェクトを設定していない場合は、[General Translationアカウント](https://wwww.generaltranslation.com/signin)にログインしてください。
*DEV* APIキーのページを開き、新しい*DEV* APIキーを作成します。
最後に、*DEV* APIキーとプロジェクトIDをコピーして、`.env`ファイルに貼り付けます。

## 結論
これで、翻訳を始める準備が整ったはずです。
あとは `<T>` コンポーネントをプロジェクトに追加して、翻訳を始めるだけです！
