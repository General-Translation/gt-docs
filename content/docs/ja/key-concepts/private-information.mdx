## 概要
個人情報は一般的に、プライベートに保たれるべき情報のことです。
例えば、ユーザーの社会保障番号（SSN）、納税者識別番号（TIN）、またはメールアドレスなどです。

翻訳中にこの情報をプライベートに保つために、`<Var>`コンポーネントを使用します。
`<Var>`コンポーネントは共有されるべきではないコンテンツをラップするために使用され、
`<Var>`コンポーネント内のコンテンツは常に同じままです。
これは名前、メール、住所などに最適です。

時には、プライベート情報を表示したいけれども、ユーザーの好みの言語や地域の規制などに基づいて再フォーマットする必要がある場合があります。
例えば、ユーザーの銀行口座残高を現地通貨で表示したい場合があります。
このような場合には、代わりに`<Currency>`、`<DateTime>`、または`<Num>`コンポーネントのいずれかを使用します。


### 例
`<Var>`コンポーネントは個人情報を表示する際にも役立ちます。

```jsx title="PrivateInfo.jsx" copy
import { T, Var } from 'gt-next'

export default function PrivateInfo(email) {
    return (
        <T id='private-info'>
            Your email address is <Var>{email}</Var>.
        </T>
    );
}
```
`<Var>`コンポーネントの子要素は翻訳されません。
そのコンテンツは決してGeneral Translation APIに渡されることはありません。
そのコンテンツはCDNに配布されることはありません。
