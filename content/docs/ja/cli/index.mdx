## 概要

General Translation CLIツール（`gtx-cli`）を使用すると、プロジェクトを[サポートされている言語](/docs/reference/supported-locales)のいずれかに自動的に翻訳できます。

このツールは[`gt-next`](/docs/react)と[`gt-react`](/docs/react)をプロジェクトで使用する際に最適なサポートを提供し、翻訳可能なコンテンツのスキャン、プロジェクトの翻訳、翻訳データJSONファイルの生成が可能です。

このCLIツールは、[next-intl](https://next-intl.dev/)や[i18next](https://react.i18next.com/)などのサードパーティi18nライブラリの翻訳生成にも使用できます。

---

プロジェクトで使用しているi18nライブラリに応じて、以下のチュートリアルのいずれかに従ってください：

* [gt-next](/docs/cli/tutorials/gt)
* [gt-react](/docs/cli/tutorials/gt)
* [next-intl](/docs/cli/tutorials/next-intl-quickstart)
* [i18next](/docs/cli/tutorials/i18next-quickstart)
* [その他](/docs/cli/tutorials/json)




## 使用方法

| コマンド | 説明 |
| --- | --- |
| [`npx gtx-cli init`](/docs/cli/init) | （オプション）プロジェクトのGT設定を構成します。 |
| [`npx gtx-cli setup`](/docs/cli/setup) | ステップバイステップのセットアップガイドで、プロジェクトを翻訳の準備をします。 |
| [`npx gtx-cli scan`](/docs/cli/scan) | プロジェクトから翻訳可能なコンテンツをスキャンします。プロジェクトの更新に繰り返し使用できます。 |
| [`npx gtx-cli translate`](/docs/cli/translate) | GT APIを通じてプロジェクトを翻訳します。 |
| [`npx gtx-cli generate`](/docs/cli/generate) | 新しい翻訳データJSONファイルを生成します。ライブラリをスタンドアロンで使用する場合に使用すべきです。 |
