## 使用方法

このコマンドを使用してプロジェクトのGT設定を初期化します。このコマンドはオプションであり、プロジェクトのルートに`gt.config.json`ファイルを作成するだけです。

```bash
npx gtx-cli init
```

## 概要

`init`コマンドはプロジェクトのGT設定を構成するのに役立ちます。

プロジェクトのルートに`gt.config.json`ファイルを作成します。

このファイルには以下の設定が含まれます：

- `defaultLocale`：プロジェクトのデフォルトロケール。
- `locales`：プロジェクトでサポートされているロケールの配列。
- `files`：翻訳したいコンテンツに関する情報を含むオブジェクトです。

`gt.config.json`ファイルについての詳細情報は、[config docs](/docs/cli/reference/config)を参照してください。
