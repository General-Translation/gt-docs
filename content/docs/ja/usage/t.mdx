## 概要

`<T>`コンポーネントは、General Translationでコンテンツを翻訳するための主要な方法です。
辞書と一緒に使用することも、単独で使用することもできます。

`<T>`コンポーネントはテキスト、HTML、JSXを翻訳するために使用できます。

必要なのは、テキストを`<T>`コンポーネントでラップするだけで、翻訳されます。

追加の設定は必要ありません！

```jsx
import { T } from 'gt-react'; // [!code highlight]

export default function MyComponent() {
  return (
    // [!code highlight]
    <T>
      <h1>List of Shakespeare Plays: </h1>
      <ul>
        <li>Hamlet</li>
        <li>A Midsummer Night's Dream</li>
        <li>Macbeth</li>
      </ul>
    </T> // [!code highlight]
  );
}
```


### アドバイス

特に国際化（i18n）に慣れていない場合は、その単純さから`<T>`コンポーネントの使用をお勧めします。
