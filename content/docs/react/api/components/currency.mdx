## Overview

The `<Currency>` component renders a formatted currency string, supporting customization such as currency type and formatting options.
The number is formatted based on the current locale and any optional parameters passed, adhering to proper international currency conventions.

This is useful for rendering localized currency values, such as prices or financial amounts.
Additionally, the contents of the `<Currency>` component will remain private.

### Returns

## Example

The `<Currency>` component can be used to display localized currency values.
It must be used inside of a `<T>` component or in a dictionary entry.

```jsx title="PriceDisplay.jsx" copy
import { T, Currency } from 'gt-react'; // [!code highlight]

/**
 * Renders the price of an item in a store.
 */
export default function PriceDisplay(item, curr) {
    return (
        <T>
            The price of the item is <Currency currency={curr}> {item.price} </Currency>. // [!code highlight]
        </T>
    );
}
```

For more details and usage examples of the `<Currency>` component and other variable components like `<Num>`, `<DateTime>`, and `<Var>`,
see the [Using Variable Components](/docs/react/reference/using-variable-components) documentation.
